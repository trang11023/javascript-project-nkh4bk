## 🛠 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/javascript-project-nkh4bk.git
   ```
2. Install dependencies (for javascript):
   ```bash
   npm install
   ```

## 🚀 Usage
Run the main script:
```bash
node index.js
```

Check the `docs/` folder for detailed guides.
